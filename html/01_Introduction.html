1. Introduction

1.1. Using this resource

This interactive resource is designed to complement the book ‘Modern Mathematics: An introduction through the LEAN theorem prover’.

The simplest way to use this book is to download it as as bundle from ADD THE URL HERE.

This text contains Lean code such as the following

#reduce lean.version

Every item of Lean code is available via a Lean file in a subfolder of the src folder of this project.

1.2. The Lean + VSCode interface

I assume you have opened this resource in Visual Studio Code (VSCode). If so, should be reading this document in the right-most pane, titled ‘Lean Documentation’.

If you haven’t already done so, read the instructions in the README.md document contained with this project.

The Lean Infoview window is in the middle pane. You’ll see feedback and other output from a running Lean file in this window. I’ll assume you have started by clicking on the start.lean file.

On the very left of the VSCode window is the sidebar. In Explorer view (which is selected if you click the top-left icon), you’ll see the file hierarchy for this project.

Try the following:

  • Click in the middle of the left-most pane, the pane that currently contains the start.lean file.

  • The Lean files associated with this documentation are in subfolders of the src folder. If you now go to the file 02_The_Lean_inferface.lean in the 01_Introduction subfolder of src, you’ll see the Lean file for this section.

In the Lean editor window, you’ll see the code below.

#reduce lean.version.snd

The Lean Infoview window will show you Lean’s output: 3. It has calcuated this from the first element (indicated by fst) of the Lean version number.

Modify the Lean code, replacing fst with snd. Is that what you expected?

If you ever accidentally close the Lean Documentation window, you can retrieve it by opening the Command Palette from the View -> Command Palette menu and searching for: ‘Lean: Open Documentation View’. Then click ‘Open documentation of current project’.

Likewise, you can display the Infoview window by searching for ‘Lean: Infoview: Display Goal’ in the Command Palette.
