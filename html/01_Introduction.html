1. Introduction¶

1.1. Using this resource¶

This interactive resource is designed to complement the book ‘Modern Mathematics: An introduction through the LEAN theorem prover’.

It is designed to be used interactively in Visual Studio Code.

The simplest way to run the project is to open a GitHub Codespace via the repository for this project.

1.2. Working with Lean in VSCode¶

I assume you have opened this resource in Visual Studio Code (VSCode). If so, should be reading this document in the right-most pane, titled ‘Lean Documentation’.

If you haven’t already done so, read the instructions in the README.md document contained with this project.

Click on the link ‘Go to source’ in the code fragment below.

#reduce lean.version

Two things should have happened: (1) the left-most editor will contain the contents of the Lean file corresponding to this section of the book and (2) the middle pane will display the Lean Infoview window. Initially, it shows nothing.

1. Click in the Lean file in the left-most pane to start Infoview. 2. Click the end of the line #reduce lean.version in the Lean window. You’ll then the output from this command in the Infoview window.

The output from #reduce lean.version is a triple of numbers that represents the Lean version number used by this project.

If desired, lock the Lean Documentation pane by clicking the three dots in the top-right of this pane then selecting lock group. Do the same for the Lean Infoview pane.

Now, we’ll apply the fst function to the triple lean.version.

#reduce lean.version.fst

If you click the end of the line #reduce lean.version.fst in the Infoview window, you wee see Lean’s output: 3. It has calcuated this from the first element (indicated by fst) of the Lean version number.

Modify the Lean code, replacing fst with snd. Is that what you expected?

If you ever accidentally close the Lean Documentation window, you can retrieve it by opening the Command Palette from the View -> Command Palette menu and searching for: ‘Lean: Open Documentation View’. Then click ‘Open documentation of current project’.

Likewise, you can display the Infoview window by searching for ‘Lean: Infoview: Display Goal’ in the Command Palette.

To navigate between chapters, either use the ‘Next’ and ‘Previous’ buttons below or scroll to the top and use the home button or the triple line menu.
