2. Expressions

2.1. Evaluating expressions

Lean can be used as a calculator to evaluate expressions.

try it!
#eval 2 * 3

The above code outputs 6 in the Infoview window.

Adapt the code to evaluate expressions such as \(1 + 2 + 3\), \(7^4\), \(6 - 3\), \(20/5\), \(10 + 2 \times 5\).

Remember that to work with the code for this section, you should first click in the left-most editor pane (which should currently display a Lean file) then select the Lean file for this section from the src folder of the file hierarchy shown in the sidebar on the left.

You may add comments to your Lean code. Lean ignores any text within a comment.

try it!
/-
This is a block comment. It is ignored by Lean
-/

#eval 1 + 2 -- This line comment starts at the `--` and ends at the end of the line.

2.2. Types in Lean

Every expression in mathematics (and in Lean) has a type.

try it!
#check 2

The code above outputs 2 : ℕ, showing that 2 has type ℕ, the type of natural numbers.
